Work

Lyrical App

Node/Express | Angular | Ruby/Sinatra | HTML5 | CSS3 | Materialize

Lyrical is a secure angular app that allows users to search, create, and save lyrics in one interface. Data is stored in relational mySQL tables and user authentication is secured with bCrypt. Lyrical's server side is built with Sinatra and Ruby and utilizes a RESTful API for user account and lyric data.

Artsy App

Node/Express | Javascript | Handlebars | CSS3 | SASS | Materialize

Artsy is an art gallery app created as part of a pair programming project. With Artsy users can upload art work via a base64 file encoder to be queried as high-quality images in user galleries. Artsy was built with Node/Express and views are rendered with Handlebars.

Get Ripped Or Die Trying

Node/Express | Javascript | Phaser.js | Handlebars | CSS3 | LESS | Materialize

Get Ripped Or Die Trying is a top-down survival game built with the Phaser.js arcade engine on Node/Express. Collision detection is set for interacting objects along with physics and seperation dynamics. Players win by collecting milk cartons and lose when collision is detected between the player and enemy sprites.
